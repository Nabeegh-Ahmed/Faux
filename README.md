# Faux
Faux is a paper trading platform where you can learn to trade with virtaul money but real market data.

# Tech Stack
<li>Next JS</li>
<li>GraphQL</li>
<li>Strapi Headless CMS</li>

# APIs Used
<li>Finnhub API for real time price data</li>
